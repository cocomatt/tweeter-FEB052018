# Tweeter Project

Tweeter is a simple, single-page Twitter clone.

A single Page App built with HTML, CSS, JS, jQuery and AJAX on the client-side and Node, Express and MongoDB on the server-side.

## Dependencies

- Express
- Node 5.10.x or above
